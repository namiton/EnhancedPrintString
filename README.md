# EnhancedPrintString
 
本プラグインはアンリアルエンジンのPrintStringを拡張しより制御しやすいようにしたものです。 
  
# Requirement
 
Windows10,Windows11の64bitでの動作を確認しています。  
 
 
# Usage

プロジェクトフォルダ直下にPlugins を作成しその中に解凍したフォルダを入れてリビルドして下さい。
問題が発生した場合はプラグインを無効化しリビルドしてください。
